# Patch 16 - Income source management

This patch adds:

- archive support for income sources
- one-time income source support
- backend filtering for archived sources
- current-month one-time income rows
- Review modal action bar rollback to the previous non-sticky behavior

After copying the patch files into the repo, run:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\apply-patch-16.ps1
```
